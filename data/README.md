# Data Folder

## Dataset: CORGIS Airlines Dataset

**Source:** CORGIS Dataset Project (Bureau of Transportation Statistics)

**Download URL:** https://corgis-edu.github.io/corgis/datasets/csv/airlines/airlines.csv

**File:** `airlines.csv` (already downloaded)

## Dataset Description

Monthly aggregated flight delay data for US airports from 2003-2016.

- **Rows:** 4,409
- **Airports:** 29 major US airports
- **Time Period:** 2003 - 2016

## Key Columns

| Column | Description |
|--------|-------------|
| Airport.Code | Airport IATA code (e.g., ATL, ORD, DFW) |
| Airport.Name | Full airport name |
| Time.Year | Year of data |
| Time.Month | Month (1-12) |
| Statistics.Flights.Total | Total flights |
| Statistics.Flights.On Time | On-time flights |
| Statistics.Flights.Delayed | Delayed flights |
| Statistics.Flights.Cancelled | Cancelled flights |
| Statistics.# of Delays.Carrier | Delays due to carrier issues |
| Statistics.# of Delays.Weather | Delays due to weather |
| Statistics.# of Delays.Late Aircraft | Delays due to late aircraft |
| Statistics.# of Delays.National Aviation System | Delays due to air traffic control |
| Statistics.# of Delays.Security | Delays due to security |
| Statistics.Minutes Delayed.* | Total delay minutes by cause |

## Data Source

Original data from Bureau of Transportation Statistics (BTS), U.S. Department of Transportation.
