1. Install packages:

2. The dataset is already downloaded in the `data/` folder (`airlines.csv`)

3. Open notebook:

## Dataset

**Source:** CORGIS Dataset Project (BTS Data)

**URL:** https://corgis-edu.github.io/corgis/datasets/csv/airlines/airlines.csv

Monthly aggregated US flight delay data from 2003-2016.
